# Content Management System for Blog

## Project Description
This project is developed as part of an Industrial Internship in collaboration with
upskill Campus and UniConverge Technologies Pvt. Ltd.

The system allows users to:
- Create and publish blog posts
- Design pages using content editors
- Store content in a database
- Serve blogs over HTTP/HTTPS

## Technologies Used
- Python (Flask)
- HTML/CSS
- SQLite
- GitHub

## How to Run
1. Install Flask
2. Run app.py
3. Open browser and go to http://127.0.0.1:5000/
